# Odin Recipe Project

The project consist of steps needed to be done using what I've learned. 

Create a page to link to other pages that consists of recipes of a dish. In the recipes page, add the name of the dish using heading (h1), image (img) of the dish, description (h2)(p) of the dish, ingredients (h2) and the list in an unordered list (ul)(li), and lastly the steps (h2) in an ordered list (ol)(li).
